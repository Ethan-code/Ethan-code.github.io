## 安裝 Electron
1. npm 初始化

  ```bash
  npm init
  ```

1. 安裝 Electron 做為開發工具

  ```bash
  npm install --save-dev electron
  ```

1. 加入 `.gitignore`，可以從 [electron/.gitignore][electron.gitignore] 上下載範本
