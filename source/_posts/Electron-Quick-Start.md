## 安裝 Electron
- npm 初始化
    ```bash
    npm init
    ```
- 安裝 Electron 做為開發工具
    ```bash
    npm install --save-dev electron
    ```
- 加入 `.gitignore`
    - 從 [electron/.gitignore][electron.gitignore] 上下載範本
