Kubuntu 安裝 Go 環境

Posted on 2021-12-21 Edited on 2022-06-06

步驟

  1. 安裝 Go 的執行環境，到官網下載頁面下載 tar.gz 檔（此次版本為 go1.17.5.linux-amd64.tar.gz）

  2. 解壓縮到 /usr/local/go 路徑下

  3. 加入 go 指令到 PATH 內

  4. 執行 go version 確認安裝沒問題

參考資料

  • Download and install - go.dev
  • Tutorial: Get started with Go - go.dev
