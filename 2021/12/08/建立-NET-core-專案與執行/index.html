建立 .NET core 專案與執行

Posted on 2021-12-09 Edited on 2021-12-18

建立專案

到要建立專案的資料夾下，執行 dotnet new console 建立專案

  • console 表明建立 主控台 應用程式，可以替換成其他 專案模板

建立 .gitignore 檔案

  • 到 core/.gitignore at main · dotnet/core 下載 .gitignore 檔案

執行

執行 dotnet run
